# Path Safety Assessment Tool (PSAT) User Guide

Welcome to the Path Safety Assessment Tool (PSAT). This tool allows you to evaluate active mobility paths using the CycleRAP model.

## 1. Getting Started
* **Start a Project**: From the Home page, click "Create Project" to import your shapefiles, mapping data, and street-level imagery.
* **Project Settings**: Ensure your project name (e.g. Road Name) is correctly set. You can insert relevant tags (e.g. Ang Mo Kio) to group your projects. Upload your images by selecting the correct local drive.
* **Select project(s)**: You can select single or manage multiple projects from the Projects listing. You can also edit the metadata (i.e. Project Name and Tags) of the project(s).
