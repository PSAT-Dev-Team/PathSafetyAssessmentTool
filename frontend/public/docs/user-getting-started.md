# Path Safety Assessment Tool (PSAT) User Guide

Welcome to PSAT. The tool helps you create projects from geotagged survey images, review coded path segments, calculate CycleRAP risk, and test treatments.

## 1. Getting Started

### Open the project list

Use the Home page to browse all projects. You can:

- search by project name or road name
- filter by tag
- sort by verification, autocode progress, and last modified time
- select one or more projects for coding, path analysis, or treatment work

### Create a project

Use Create Project when you want to build a new project from the source image folders in `in/`.

You can create a project in two ways:

1. select one source folder directly
2. draw a polygon or click a planning area on the map and create a project from multiple selected roads

When creating a project:

- enter a project name without underscores
- add tags if needed
- upload images into a source folder if the folder is missing
- check that selected roads are marked as available before you create the project

### Navigate between workflows

From the project list, you can send selected projects to:

- Coding for detailed review and saving
- Analyse Projects for multi-project filtering, charts, and exports
- Treatment Application for before/after scenario testing

### Use Help and GIS Layers

- use the Help button for the in-app user and developer guides
- use View GIS Layers from the sidebar when you need to inspect the GIS data behind the coding rules
