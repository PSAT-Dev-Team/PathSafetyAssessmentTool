## 4. Path Analysis
The Path Analysis module allows users to analyze projects and evaluate the safety and risk levels of path segments using the CycleRAP scoring model.
* **Project Selection**: You can select one or more projects to analyze. 
* **Filter Projects**: You can filter projects based on: Project Name, Date Created, Last Updated, Tags.
* **Filter Segment**: You can apply up to 5 filters to visualize segments with specific attributes. This allows you to focus on segments that meet certain conditions. 
* **Risk Assessment**: The system evaluates segment attributes (e.g. line of sight, path width, surface condition) to determine risk levels for different crash types: Bicycle-Bicycle, Bicycle-Pedestrian, Single Bicycle, Vehicle-Bicycle.
* **Segment Visualization**: Filtered segments are displayed based on the selected criteria, allowing users to identify patterns and higher-risk areas.
* **Validation**: User can toggle back to the coding sheet and update the parameters.
* **Data export**: User can export the filtered data into CSV file.
