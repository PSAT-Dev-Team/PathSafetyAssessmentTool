## 4. Path Analysis

The Path Analysis page is the multi-project analysis workspace.

### Select projects

You can select one or more projects to analyze together. The search box also matches source-road names, so you can find projects even when the project title and road name differ.

### Filter before loading

Before loading projects, you can filter by:

- project or road name
- tag
- date created range
- last updated range

### Analyze loaded data

After loading projects, you can:

- choose up to five attributes to focus on
- apply segment-level filters
- view attribute distributions and aggregated score-band summaries
- inspect filtered segments on the map and in the table

### Export

The page supports two main exports:

- Download Table for the current filtered table as CSV
- Download Images for the current filtered images as a ZIP file

### Session continuity

Your selections and filters are kept for the browser session, so navigating away and returning does not immediately clear the analysis setup.
